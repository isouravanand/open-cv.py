## 📌 Getting Started
### 🔧 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/opencv-python.git
   cd opencv-python
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
